# k-max Distance

The k-max distance for images.
The repository contains two sample codes:
- main.cpp shows computing using our *Image* structure and *std::vector*
- main2.cpp shows computing using C arrays

## Requirements

* c++ std11
* opencv

## Usage

    make
    ./main or ./main2
